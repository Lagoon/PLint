h2. <a>showApp</a>

Method to get application information.
Returns a JsonElement with application and environment attributes.

*Output*

bc. {
     "notify": true,
     "activationUrl": "",
     "description": "",
     "name": "xcoisas_env",
     "url": "",
     "application": {
        "description": "buy and sell your products",
        "name": "xcoisas" 
    }
}
