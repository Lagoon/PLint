h2. <a>updateContext</a>

Method to update a context.
Returns: JsonElement with information about updated context

*Input* 

bc. name: context name
 activationUrl: url for users password registry
 url: context url
 description: context description
 oldname: name of the context to update

*Output*

bc. {
    "id": 831,
    "defaultContext": false,
    "name": "edited",
    "description": "edited context description",
    "profiles": [
        {
            "id": 1,
            "defaultProfile": false,
            "description": "admin profile",
            "name": "admin"
        },
        {
            "id": 2,
            "defaultProfile": false,
            "description": "",
            "name": "test"
        },
        {
            "id": 3,
            "defaultProfile": true,
            "description": "",
            "name": "guest"
        }
    ]
}
