h2. <a>createContext</a>

Method to create a new context.
Returns: JsonElement with information about created context

*Input* 

bc. name: context name
 activationUrl: url for users password registry
 url: context url
 description: context description
 copyContext: copy default context configuration if true

*Output*

bc. {
    "id": 831,
    "defaultContext": false,
    "name": "new",
    "description": "new context",
    "profiles": [
        {
            "id": 1,
            "defaultProfile": false,
            "description": "admin profile",
            "name": "admin"
        },
        {
            "id": 2,
            "defaultProfile": false,
            "description": "",
            "name": "test"
        },
        {
            "id": 3,
            "defaultProfile": true,
            "description": "",
            "name": "guest"
        }
    ]
}
